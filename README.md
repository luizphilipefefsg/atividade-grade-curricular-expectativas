Grade Curricular e Perspetivas Profissionais

Este repositório contém a atualização da atividade prática de desenvolvimento web realizada no dia **04/05/2026**, como parte do curso Técnico em Informática. O foco desta etapa foi a implementação de estruturas de dados tabulares em HTML e a elaboração de um texto dissertativo-crítico sobre a experiência académica.

## 📋 Requisitos Implementados

A página `index.html` foi atualizada com a inclusão de novas secções delimitadas por linhas horizontais (`<hr>`):

1. **Grade Curricular (`<table>`)**:
   * Construção de uma tabela detalhando os módulos e as disciplinas estudadas até o momento (como Lógica com Python, Desenvolvimento Web, POO com C#, SQL e Análise de Sistemas).
   * Uso de tags semânticas estruturais: `<thead>`, `<tbody>`, `<tr>`, `<th>` e `<td>`.
   * Aplicação de atributos de formatação nativos (`border`, `cellpadding`, `cellspacing` e `bgcolor`).

2. **Resumo e Expectativas (`<p>`)**:
   * Texto autoral descrevendo as expectativas iniciais ao ingressar no curso técnico.
   * Análise do alcance dos objetivos com as matérias atuais e a transição para Programação Orientada a Objetos (POO).
   * Projeção de carreira e intenção de atuação no mercado de desenvolvimento de software e banco de dados.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica e exibição dos dados.
* **Git & GitHub**: Controle de versão e envio da entrega dentro do prazo estipulado.
